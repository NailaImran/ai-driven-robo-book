# QuantumPages 🚀

A modern, interactive learning platform for Physical AI, Robotics, and Advanced Physics concepts.

## Features ✨

- **📚 Comprehensive Curriculum**: 6 chapters covering physics, AI, robotics, and more
- **🌍 Bilingual Support**: Content available in English and Urdu
- **💻 Interactive Examples**: Code samples, visualizations, and practical implementations
- **🎨 Modern Design**: Built with Next.js and Tailwind CSS for a beautiful user experience
- **📱 Responsive**: Works seamlessly on desktop, tablet, and mobile devices
- **⚡ Fast & Optimized**: Next.js for optimal performance

## Getting Started 🛠️

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd quantum-pages
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Project Structure 📁

```
quantum-pages/
├── app/              # Next.js app directory
│   ├── layout.tsx    # Root layout
│   ├── page.tsx      # Homepage
│   ├── chapters/     # Chapter pages
│   └── about/        # About page
├── components/       # React components
├── lib/             # Utilities and data
├── types/           # TypeScript types
├── styles/          # Global styles
├── public/          # Static assets
└── tailwind.config.ts
```
