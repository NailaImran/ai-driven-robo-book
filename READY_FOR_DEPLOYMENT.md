## 📞 Deployment Support

### If deployment fails:
1. Check GitHub Actions logs
2. Review error messages
3. Verify all files are committed
4. Ensure no merge conflicts
5. Check `docusaurus.config.ts` configuration

### If site doesn't update:
1. Clear browser cache (Ctrl+Shift+Delete)
2. Hard refresh (Ctrl+Shift+R)
3. Wait 5 minutes for GitHub Pages cache to clear
4. Verify workflow completed with green checkmark

### If styles are broken:
1. Verify `baseUrl: '/'` in docusaurus.config.ts
2. Check CSS files in build output
3. Verify CSS modules are compiled
