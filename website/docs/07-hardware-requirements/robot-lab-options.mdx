# Robot Lab Infrastructure

Design and build a complete robotics laboratory for research, education, or commercial development.

---

## Lab Planning

### Space Requirements

**Minimum Lab** (Academic/Small Team):
- **Floor space**: 20m² (4m x 5m)
- **Ceiling height**: 2.5m minimum
- **Power**: 20A dedicated circuit
- **Network**: Gigabit Ethernet, Wi-Fi 6

**Professional Lab** (Research/Industry):
- **Floor space**: 100m² (10m x 10m)
- **Ceiling height**: 3.5m (for overhead sensors/cameras)
- **Power**: 50A three-phase
- **Network**: 10 Gigabit fiber backbone

## Essential Equipment

### 1. Robotics Testing Area

**Safety Enclosure** ($2,000-10,000):
```text
Purpose: Contain autonomous robots during testing
Options:
  - Basic: Portable barriers (safety tape + plastic barriers)
  - Mid-tier: Modular fence panels (aluminum, 2m high)
  - Professional: Industrial safety cage with interlocked gates
```

**Floor Surface**:
```text
Recommended: Smooth concrete or industrial vinyl
Avoid: Carpet (interferes with wheels), glossy tile (LiDAR reflections)
Markings: Paint grid lines (1m squares) for calibration
```

**Overhead Camera System** ($1,500-5,000):
```text
Purpose: Ground truth position tracking (motion capture lite)
Components:
  - 4-8x USB cameras (1080p, wide-angle)
  - Ceiling mounts
  - AprilTag markers on robots
  - Computer running OpenCV tracking
```

**Electronics Workstation** ($1,500):

- ESD-safe workbench (1.5m x 0.75m)
- Magnifying lamp

**Mechanical Workshop** ($2,500):

- Drill press

Materials:

## Networking Architecture

### Network Topology

```
                   Internet
                      │
                  [Router]
                      │
              [Managed Switch]
         ┌────────┼────────┬────────┐
         │        │        │        │
    Workstation Robot    Robot    NAS
       Cluster   Fleet    WiFi   Storage
```

### VLAN Configuration

```bash
# VLAN 10: Development workstations
# VLAN 20: Robot control network (isolated)
# VLAN 30: Sensor data network (high bandwidth)
# VLAN 40: Guest Wi-Fi (internet only)

# Example switch config (pseudo-code):
vlan 10
  name DEV_WORKSTATIONS
  ports 1-8

vlan 20
  name ROBOT_CONTROL
  ports 9-16
  priority high

vlan 30
  name SENSOR_DATA
  ports 17-20
  bandwidth 10Gbps
```

### ROS 2 Multi-Machine Setup

```bash
# On each machine, set ROS_DOMAIN_ID
export ROS_DOMAIN_ID=42

# Enable multicast (required for DDS discovery)
sudo ip route add 224.0.0.0/4 dev eth0

# For large networks, use Fast DDS discovery server
# On server machine:
fastdds discovery -i 0

# On client machines:
export ROS_DISCOVERY_SERVER=192.168.1.100:11811
```

### Safety Procedures

1. **Pre-Test Checklist**:
   - [ ] Testing area clear
   - [ ] E-stop functional
   - [ ] Observers present (minimum 2 people)
   - [ ] Robot battery charged
   - [ ] Software kill switch tested

2. **During Testing**:
   - One person as "safety officer" with e-stop
   - No one inside test area during autonomous operation
   - Monitor battery levels

3. **Incident Reporting**:
   - Log all collisions, near-misses, software crashes
   - Review weekly for patterns

## Environmental Controls

### Climate

```text
Temperature: 20-25°C (68-77°F)
Humidity: 40-60% RH
  - Too dry: Static electricity damages electronics
  - Too humid: Corrosion, condensation

Solutions:
  - HVAC with humidity control
  - Dehumidifier in summer
  - Anti-static mats at workbenches
```

### Lighting

```text
Overhead: 500 lux (bright, uniform)
Workbench: 1000 lux (task lighting)
Options:
  - LED panels (energy efficient, no flicker)
  - Adjustable desk lamps
  - Avoid fluorescent (can interfere with cameras)
```

### Data Management

```bash
# Automated backup script (run daily via cron)
#!/bin/bash
# backup_lab_data.sh

DATE=$(date +%Y%m%d)
BACKUP_DIR="/mnt/nas/backups/$DATE"

# Backup ROS bags
rsync -av /home/*/ros2_ws/logs/ $BACKUP_DIR/ros_logs/

# Backup datasets
rsync -av /datasets/ $BACKUP_DIR/datasets/

# Backup git repositories
find /home/*/ros2_ws/src -name .git -execdir git bundle create $BACKUP_DIR/repos/{}.bundle --all \;

# Backup system configs
tar -czf $BACKUP_DIR/configs.tar.gz /etc/ros/ ~/.bashrc ~/.config/

# Clean old backups (keep 30 days)
find /mnt/nas/backups/ -mtime +30 -delete
```
