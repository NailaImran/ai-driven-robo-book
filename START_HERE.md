## 🚀 Common Tasks

### View the Website
```bash
cd quantum-pages
npm install --legacy-peer-deps
npm run dev
# Open http://localhost:3000
```

### Add a New Chapter
1. Edit: `quantum-pages/lib/chapters.ts`
2. Add new chapter object to array
3. Changes appear instantly!

### Change Colors
1. Edit: `quantum-pages/tailwind.config.ts`
2. Modify the `quantum` color palette
3. Save and refresh browser

### Deploy to Web
- **Vercel**: Connect GitHub repo (easiest!)
- **Netlify**: Upload `.next` folder
- **Docker**: Use provided Dockerfile template
- **Server**: Use `npm run build && npm run start`
